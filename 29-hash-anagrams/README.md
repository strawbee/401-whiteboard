# Whiteboard 29 - Hash Map of Anagrams
Joy Hou, Feb 22, 2018

## Team Members
Bill

## Problem Domain
Write a function that takes in a string and returns a hash map of all the anagrams in the string. Example:

```sortAnagrams('acre race care act cat tac');```

Returns:
```
{
    'acer': ['acre', 'race', 'care'], 
    'act': ['act', 'cat', 'tac']
};
```
